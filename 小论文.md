#  ##领域专利知识图谱构建研究*

<span style="font-size:10.5pt">^1^鱼宝银</span>

<span style="font-size:10.5pt">^1^(兰州理工大学机电工程学院  兰州 730050)</span>

<span style="font-size:10.5pt">^2^(兰州理工大学建筑与艺术设计学院  兰州 730050)</span>

<span style="font-size:10.5pt">**摘要**：【目的】为提升垂直领域知识驱动的产品设计效率及构建有效的领域知识数据库，提出一种基于混合实体分类识别网络和大语言模型驱动的专利知识图谱构建解决方案。【方法】混合Bert、Roberta、Erine网络，采取预训练加微调的策略抽取专利权利要求书中的实体，在此基础之上通过对齐实体识别结果及专利文本数据集，实现实体数据的分割标记，以此构建大语言模型提示词模版，实现关系三元组抽取。【结果】以创建的无人机专利数据集进行实验，实体识别取得了96.9%的准确率和89.5% 的F1值，关系抽取获得了xx%和xx%。【局限】【结论】所提方法能够高效、准确地抽取专利中的设计知识，为领域知识图谱的高效构建及基于知识驱动的产品创新设计提供了一种切实可行的方法。</span>

<span style="font-size:10.5pt">**关键词：**领域知识图谱；专利；信息抽取；深度学习；产品设计</span>
